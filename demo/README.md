# 미진전기 긴급 출동 디스패치 — 계약 시연용 데모

> 계약 미팅에서 고객사(미진전기)에게 보여줄 **클릭형 화면 시안**입니다.
> 실제 백엔드/데이터는 없고, 흐름과 핵심 가치를 보여주는 **하드코딩 데모**입니다.

## 여는 법
- `demo/index.html` 파일을 **더블클릭** → 브라우저에서 **풀스크린 앱**으로 바로 열립니다.
- **인터넷 불필요** (외부 라이브러리 0개). 노트북·태블릿·폰 모두 가능.
- 하단 바의 **`다음 →`** 버튼을 누르면 **순서대로** 진행됩니다 (홈 → 신고 → 자동매칭 → 🏢 업체 수락 → 출동 → 완료 → 🛠️ 관리자). `← 이전`으로 되돌아갈 수 있고, 가운데에 현재 단계(예: `출동업체 · 수락 4/7`)가 표시됩니다.
- 특정 화면으로 바로 점프하려면 하단 왼쪽 **`≡` 버튼**(목차)을 누르세요.
- `≡` → **`⛶ 전체화면`** 을 누르면 브라우저 UI까지 사라진 **진짜 전체화면**으로 시연할 수 있습니다 (계약 자리 추천).

## 화면 구성 (하단 `다음 →` 으로 순서 진행, `≡` 로 목차 점프 — 총 7단계)

1. **홈** — 미진 브랜드 + "초긴급 1시간 / 긴급 2시간" 약속을 첫 화면에서 각인
2. **신고서** — 전기 고장 수리요청 (위치·고장유형 6종·긴급도·상세증상·사진)
3. **자동매칭** ⭐ — 긴급도에 따라 **적합 업체 범위가 실제로 달라지는** 핵심 화면
4. **🏢 업체 수락** (출동업체 관점) — 접수함: 가용 토글 + 새 제안(긴급도·거리·SLA 마감·**응답 카운트다운**) → 수락/거절 → **소속 기사 지정**
5. **출동 현황** — 배정 업체/기사, 진행 타임라인, 기사 '도착'·'완료' 버튼
6. **완료** — SLA 준수 결과 요약 (결제는 현장 대면 = 플랫폼 미중개 명시)
7. **🛠️ 관리자 콘솔** (미진 관점) — KPI(진행중·임박·위반·승인대기) + 실시간 SLA 모니터링 + **수동 재배정** + **가입 승인 대기열**

## 추천 시연 순서 (3분 시나리오 · `다음 →` 으로 진행)
1. **홈** 에서 "1시간/2시간 약속"을 말로 한 번 짚기.
2. **신고** 화면 → 고장유형 `누전·차단기`, 긴급도 **🔴 초긴급** 선택.
3. **자동매칭** → "초긴급이라 **도착 60분 이내** 업체 3곳만 후보가 되고, 가장 빠른 한빛전기설비가 자동 배정됩니다" 설명.
   - 👉 **킬러 포인트:** `← 이전`으로 신고로 돌아가 긴급도를 **🟠 긴급**으로 바꾼 뒤 다시 매칭 → **후보가 5곳으로 늘어남.** "긴급은 2시간이라 더 멀리까지 매칭" — SLA·거리 연동을 눈으로.
4. **🏢 업체 수락** → "방금 그 신고를 배정받은 출동업체 화면입니다" — 응답 카운트다운 + 수락 → 기사 지정으로 **출동업체 관점** 보여주기.
5. **출동 현황** → '도착' → '완료' 탭하며 SLA 시계 정지/종료 시연.
6. **완료** 화면의 "SLA 충족 / 시한 대비 여유" 로 고객 흐름 마무리.
7. **🛠️ 관리자 콘솔** → 위반 건 **수동 재배정** + 가입 **승인**으로 "미진이 모든 걸 통제한다"를 어필.

## 데모에 담긴 핵심 메시지 (고객사 설득 포인트)
- **SLA-거리 연동 자동매칭**: 초긴급=1시간권, 긴급=2시간권 업체만 후보 → 약속을 지킬 수 있는 구조라는 신뢰.
- **풍부한 전기고장 신고서**: 누전·정전·합선·분전반·콘센트·동력설비 등 현장 언어로 구성.
- **거절·무응답 시 자동 재배정** 문구로 SLA 안전망 어필.
